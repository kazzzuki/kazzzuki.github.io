    Learnings and Reflections

    Throughout my journey in CPE232 - Managing Enterprise Servers, I’ve learned a lot and gained new knowledge about the different topics discussed in this course. Starting with the exploration of DevOps, this course helped me understand the concept of this software. It is the combination of practices and tools that are designed to improve and increase the organization’s ability to serve applications and different services faster than the other traditional software development processes.

    The modules on SSH-key-based authentication and the installation of SSH server on CentOS are both new to me; we did not do that in Track Elective 1. With These topics retained in my mind, I understand its concept and importance about securing server access and server setup. Now, with Ansible, this tool was introduced in Module 4. I learned and observed that this tool is very powerful in terms of automating tasks and optimizing scalability. Also, the creation of roles in Ansible, provides an approach to target server configuration, enhancing efficiency and maintainability.

    The containerization concept and the Docker in Module 11 gives me an understanding of its efficiency and portability in different containerized environments. I also learned to build applications using this concept and it is very ideal in deployment processes. This course also taught me about the different cloud services, particularly OpenStack. I learned about the scalable and flexible services offered by OpenStack for cloud infrastructure.

    In conclusion, CPE232 has been a fun exploration, from the fundamental concepts to different advanced cloud services, and providing a well-rounded skills set in system administration. The hands-on-activities enhanced my knowledge about the topics per module and especially in real-world scenarios. My most memorable experience in this course is when I encountered an error that I encountered for the first time because troubleshooting really helps me understand the topic even more compared to not encountering any error because I know that encountering these errors will help me in the future since I already know how to solve them.

    Overall, this course has been fun, although we were not able to use the new classroom and PCs.
